QuAnGIS vocabularies

During development, QuAnGIS RDF files are published on a GitHub Page. These are the latest versions; see also http://geographicknowledge.de/vocab/.

  • https://quangis.github.io/vocab/cct: Ontology for the core concept transformation language.
  • https://quangis.github.io/vocab/tool: Tool schema.
  • https://quangis.github.io/tool/repo.ttl: Tool repository.
  • http://geographicknowledge.de/vocab/CoreConceptData.rdf: Core concept datatypes.
